CSSnotes

This is a website made for students to gain knowledge about basic css. In this webiste css topics sre subdivided into 4 subunits. 
1. Intro and types of CSS
2. Selectors, Properties and units
3. Animation, Transition and Hover
4. flex, Grid, position & responsive
