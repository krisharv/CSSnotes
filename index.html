3.1

Introduction to CSS & Types of CSS

What CSS is, Why we use it, and the three ways to add it to a webpage

What is CSS

Defination

CSS (Cascading Style Sheets) is a stylesheet language used to control the visual appearance of the HTML elements on a webpage. While HTML defines the structure and content (what is there), CSS defines the presentation (how it looks)- colours, fonts, sizes, spacing, layouts, animations and more. CSS works by selecting HTML element and applying Style rules to them

HTML Does CSS does
Defines structure (headings, paragraphs, links) Definess appearance (colour, font, size, layout)
<h1>Hello</h1> - puts text on page Makes that h1 red, 32px, bold, centered
Content skeleton The skin, makeup and outfit
📌 The word "Cascading" means styles flow down from parent to child elements, and when multiple rules target the same element, a specificity priority order decides which one wins

CSS Syntax

Syntax

A CSS Rule has two parts: a selector ( which HTML element to target) and a declaration block(what to change). Inside the block, each declaration is a property-value pair seperated by a colon, ending with a semicolon. 

/*Basic CSS syntax*/
selector{
    property: value;
    property: value;
}

/* Real example */
h1{
    color: red;
    font-size: 32px
    text-align: center;
}

Three types of CSS

Overview

There are three ways to add CSS to a web page: Inline CSS (directly on an element), Internal CSS (in a <style> tag in the <head>), and External CSS (in a seperate .css file linked to the HTML). Each has different scope, priority, and best use situation.

1.Inline CSS

Defination

Inline CSS is written directly inside an HTML element using the styleattribute. It applies styles to only that specific element. It has the highest priority - it overrides both internal and external CSS. However, it mixes content with styling and is very hard to maintain at scale.

<!-- Inline: style="" directly on the element -->
<h1 style="color:blue; font-size: 32px; text-align: center;">Hello World</h1>

<p style="color: gray; font-family: Arial; margin: 10px;">
  Paragraph with inline styles.
</p>
