What is CSS

Defination

CSS (Cascading Style Sheets) is a stylesheet language used to control the visual appearance of the HTML elements on a webpage. While HTML defines the structure and content (what is there), CSS defines the presentation (how it looks)- colours, fonts, sizes, spacing, layouts, animations and more. CSS works by selecting HTML element and applying Style rules to them

HTML Does CSS does
Defines structure (headings, paragraphs, links) Definess appearance (colour, font, size, layout)
<h1>Hello</h1> - puts text on page Makes that h1 red, 32px, bold, centered
Content skeleton The skin, makeup and outfit
📌 The word "Cascading" means styles flow down from parent to child elements, and when multiple rules target the same element, a specificity priority order decides which one wins
